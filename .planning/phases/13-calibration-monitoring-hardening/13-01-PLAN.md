  2. Add `cameo_root_code` column to `Prediction` model: `Mapped[Optional[str]] = mapped_column(String(4), nullable=True, index=True)`. This stores the GDELT EventRootCode (e.g. "14") for per-CAMEO weight lookup at prediction time.

  3. Add `CalibrationWeightHistory` model:
     - `id`: Integer primary key autoincrement
     - `cameo_code`: String(30) NOT NULL
     - `alpha`: Float NOT NULL
     - `sample_size`: Integer NOT NULL
     - `brier_score`: Float nullable
     - `computed_at`: DateTime(timezone=True) NOT NULL default _utcnow
     - `auto_applied`: Boolean NOT NULL default True
     - `flagged`: Boolean NOT NULL default False
     - `flag_reason`: Text nullable
     - Indexes: `ix_cwh_computed` on computed_at, `ix_cwh_cameo` on cameo_code

  4. Add `PolymarketComparison` model:
     - `id`: Integer primary key autoincrement
     - `polymarket_event_id`: String(100) NOT NULL
     - `polymarket_slug`: String(200) NOT NULL
     - `polymarket_title`: Text NOT NULL
     - `geopol_prediction_id`: String(36) NOT NULL index
     - `match_confidence`: Float NOT NULL
     - `polymarket_price`: Float nullable (latest Yes price 0.0-1.0)
     - `geopol_probability`: Float nullable (ensemble probability)
     - `last_snapshot_at`: DateTime(timezone=True) nullable
     - `status`: String(20) NOT NULL default "active" (active | resolved)
     - `polymarket_outcome`: Float nullable (0 or 1 when resolved)
     - `geopol_brier`: Float nullable
     - `polymarket_brier`: Float nullable
     - `resolved_at`: DateTime(timezone=True) nullable
     - `created_at`: DateTime(timezone=True) NOT NULL default _utcnow
     - Index on status, index on geopol_prediction_id

  5. Add `PolymarketSnapshot` model:
     - `id`: Integer primary key autoincrement
     - `comparison_id`: Integer NOT NULL (FK to polymarket_comparisons.id)
     - `polymarket_price`: Float NOT NULL
     - `geopol_probability`: Float NOT NULL
     - `captured_at`: DateTime(timezone=True) NOT NULL default _utcnow
     - Index on comparison_id
     - Use `ForeignKey("polymarket_comparisons.id")` for the FK constraint
