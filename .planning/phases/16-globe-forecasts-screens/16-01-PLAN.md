    **2a. Add flyToCountry() public method:**

    ```typescript
    /**
     * Animate camera to center on the given country.
     * Uses maplibre-gl flyTo with centroid from country-geometry.
     */
    public flyToCountry(iso: string): void {
      if (!this.map) return;
      const centroid = countryGeometry.getCentroid(iso.toUpperCase());
      if (!centroid) return;
      this.map.flyTo({
        center: [centroid[0], centroid[1]],
        zoom: 4.5,
        duration: 800,
        essential: true,
      });
    }
    ```

    **2b. Add setLayerVisible() public method:**

    ```typescript
    /**
     * Set visibility of a specific layer. External callers (LayerPillBar)
     * use this instead of the removed built-in toggle panel.
     */
    public setLayerVisible(layerId: LayerId, visible: boolean): void {
      this.layerVisible[layerId] = visible;
      this.rebuildLayers();
    }
    ```

    **2c. Add setLayerDefaults() public method:**

    ```typescript
    /**
     * Set initial visibility for all layers at once.
     * Globe screen uses this to set choropleth + markers ON, others OFF.
     * Dashboard screen doesn't call this (keeps existing all-ON defaults).
     */
    public setLayerDefaults(defaults: Partial<Record<LayerId, boolean>>): void {
      for (const [id, visible] of Object.entries(defaults)) {
        if (id in this.layerVisible) {
          this.layerVisible[id as LayerId] = visible as boolean;
        }
      }
      this.rebuildLayers();
    }
    ```

    **2d. Add getLayerVisible() public method (for LayerPillBar to read state):**

    ```typescript
    /** Get current visibility of a layer. */
    public getLayerVisible(layerId: LayerId): boolean {
      return this.layerVisible[layerId];
    }
    ```

    Add `export type { LayerId };` so that external components (LayerPillBar) can reference the layer ID type.

    Also export the `LAYER_IDS` constant so LayerPillBar can iterate over available layers:
