1. **BaselineCountryRisk** (`baseline_country_risk` table):
   - `id: Mapped[int]` primary key autoincrement
   - `country_iso: Mapped[str]` String(2), unique, indexed, NOT NULL
   - `baseline_risk: Mapped[float]` Float, NOT NULL (0-100 composite score)
   - `gdelt_score: Mapped[float]` Float, NOT NULL (component breakdown)
   - `acled_score: Mapped[float]` Float, NOT NULL
   - `advisory_score: Mapped[float]` Float, NOT NULL
   - `goldstein_score: Mapped[float]` Float, NOT NULL
   - `advisory_level: Mapped[int]` Integer, NOT NULL, default=1
   - `gdelt_event_count: Mapped[int]` Integer, NOT NULL, default=0
   - `acled_event_count: Mapped[int]` Integer, NOT NULL, default=0
   - `disputed: Mapped[bool]` Boolean, NOT NULL, default=False
   - `computed_at: Mapped[datetime]` DateTime(timezone=True), NOT NULL, default=_utcnow
   - UPSERT semantics: unique constraint on country_iso

2. **HeatmapHexbin** (`heatmap_hexbins` table):
   - `id: Mapped[int]` primary key autoincrement
   - `h3_index: Mapped[str]` String(20), NOT NULL, indexed (H3 cell index string)
   - `weight: Mapped[float]` Float, NOT NULL (aggregated event weight)
   - `event_count: Mapped[int]` Integer, NOT NULL
   - `computed_at: Mapped[datetime]` DateTime(timezone=True), NOT NULL, default=_utcnow
   - Index on computed_at for staleness queries

3. **CountryArc** (`country_arcs` table):
   - `id: Mapped[int]` primary key autoincrement
   - `source_iso: Mapped[str]` String(2), NOT NULL
   - `target_iso: Mapped[str]` String(2), NOT NULL
   - `event_count: Mapped[int]` Integer, NOT NULL
   - `avg_goldstein: Mapped[float]` Float, NOT NULL (sentiment indicator)
   - `computed_at: Mapped[datetime]` DateTime(timezone=True), NOT NULL, default=_utcnow
   - Composite index on (source_iso, target_iso)

4. **RiskDelta** (`risk_deltas` table):
   - `id: Mapped[int]` primary key autoincrement
   - `country_iso: Mapped[str]` String(2), NOT NULL, indexed
   - `current_risk: Mapped[float]` Float, NOT NULL
   - `previous_risk: Mapped[float]` Float, NOT NULL
   - `delta: Mapped[float]` Float, NOT NULL (current - previous)
   - `computed_at: Mapped[datetime]` DateTime(timezone=True), NOT NULL, default=_utcnow

5. **TravelAdvisory** (`travel_advisories` table):
   - `id: Mapped[int]` primary key autoincrement
   - `country_iso: Mapped[str]` String(2), NOT NULL, indexed
   - `source: Mapped[str]` String(20), NOT NULL ('us_state_dept' or 'uk_fcdo')
   - `level: Mapped[int]` Integer, NOT NULL (1-4)
   - `updated_at: Mapped[datetime]` DateTime(timezone=True), NOT NULL, default=_utcnow
   - Unique constraint on (country_iso, source) -- UPSERT on this pair
