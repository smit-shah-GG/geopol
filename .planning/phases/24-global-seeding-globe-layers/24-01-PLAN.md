1. Create `src/seeding/fips.py`:
   - Module-level `_load_fips_mapping()` that reads `src/seeding/data/fips_to_iso.csv` via `csv.DictReader`. Path resolved relative to __file__.
   - Module-level singleton `FIPS_TO_ISO: dict[str, str]` loaded at import time.
   - `fips_to_iso(code: str) -> str | None` function that:
     - Handles 2-letter codes: lookup in FIPS_TO_ISO dict
     - Handles 3-letter codes: use `pycountry.countries.get(alpha_3=code).alpha_2`
     - Returns None for unmapped codes (log warning on first occurrence)
   - `get_sovereign_isos() -> set[str]` function that:
     - Iterates `pycountry.countries` to build set of alpha_2 codes
     - Adds "XK" (Kosovo) manually
     - Returns the set (~250 codes including territories)
   - NO static dicts with country names or codes. All data comes from CSV or pycountry.

2. Create `src/seeding/population.py`:
   - Small `_POPULATION_OVERRIDES` dict (3-5 entries max: TW=23900000, EH=600000, VA=800). This is safe for content filters.
   - `get_population(iso: str) -> int` function:
     - Checks overrides first
     - Falls back to `pypopulation.get_population(iso.upper())`
     - Returns 1 for unknown (prevents division by zero)

3. Create `src/seeding/baseline_risk.py`:
   - Constants: `WEIGHTS = {"advisory": 0.35, "acled": 0.25, "gdelt": 0.25, "goldstein": 0.15}`
   - Constants: `ADVISORY_FLOORS = {4: 70.0, 3: 45.0}`
   - `compute_baseline_risk(gdelt_event_count, population, acled_fatalities, acled_event_count, advisory_level, avg_goldstein) -> float` function:
     - GDELT score: `min(100, (gdelt_event_count / (population / 1_000_000)) * 2.0)` (per-capita normalization)
     - ACLED score: `min(100, fatalities * 5.0 + event_count * 2.0)`
     - Advisory score: map levels 1->10, 2->35, 3->60, 4->90 (small dict, safe for filters)
     - Goldstein score: `max(0, min(100, (10.0 - avg_goldstein) * 5.0))` (lower Goldstein = higher risk)
     - Composite: weighted sum of 4 components
     - Apply advisory hard floors: Level 4 >= 70, Level 3 >= 45
     - Return `round(min(100, max(0, max(composite, floor))), 1)`
   - `decay_weight(age_days: float, half_life: float = 30.0) -> float` utility function:
     - Returns `math.exp(-0.693 * age_days / half_life)`
     - Used by heatmap binner, arc builder, and baseline risk aggregation
  </action>
