1. Create `src/seeding/heatmap_binner.py`:
   - `HEATMAP_RESOLUTION = 3` constant (H3 resolution 3 = ~9,229 km^2 per hex, 41,162 total cells)
   - `bin_events_to_h3(events: list[dict], resolution: int = HEATMAP_RESOLUTION, decay_half_life: float = 30.0) -> list[dict]`:
     - Iterates events, skips those without lat/lon
     - Converts each (lat, lon) to H3 index via `h3.latlng_to_cell(lat, lon, resolution)`
     - Computes weight: `severity * mentions_norm * decay_weight` where:
       - severity = abs(goldstein_scale or 0) / 10.0
       - mentions_norm = min(num_mentions or 1, 100) / 100.0
       - decay_weight = from baseline_risk.decay_weight(age_days)
     - Aggregates weights and event counts per H3 index
     - Returns list of `{"h3_index": str, "weight": float, "event_count": int}`

2. Create `src/seeding/arc_builder.py`:
   - `extract_bilateral_arcs(events: list[dict], fips_fn, top_n: int = 50) -> list[dict]`:
     - Groups events by bilateral country pairs using actor1_code's country component + country_iso
     - Actor1 country code extraction: GDELT actor codes like "USAGOV" have the first 3 chars as ISO alpha-3. Try `fips_fn()` on first 2-3 chars of actor1_code to extract country. If actor1_code is already a country code (2-3 letters), use directly. Otherwise, skip.
     - Filters: skip domestic events (same source and target country), skip events where either country is missing
     - For each bilateral pair (canonical order: min_iso, max_iso):
       - Count events
       - Sum Goldstein scores
       - Compute avg_goldstein = goldstein_sum / count
     - Sort by event_count descending, take top_n
     - Return list of `{"source_iso": str, "target_iso": str, "event_count": int, "avg_goldstein": float}`

3. Create `src/seeding/risk_delta.py`:
   - `compute_risk_deltas(current_risks: dict[str, float], previous_risks: dict[str, float], threshold: float = 10.0) -> list[dict]`:
     - For each country in current_risks, compute delta = current - previous (0 if not in previous)
     - Filter to |delta| >= threshold (significant changes only)
     - Return list of `{"country_iso": str, "current_risk": float, "previous_risk": float, "delta": float}`
  </action>

   - `async def compute_all_layers() -> dict[str, int]`:
     a. Initialize DB: `from src.db import postgres; if postgres.async_session_factory is None: postgres.init_db()`
     b. Load events from SQLite: instantiate EventStorage, query recent events (90-day window for baseline risk, 30-day for heatmap). Include lat, lon, country_iso, goldstein_scale, num_mentions, actor1_code, event_date, source fields.
     c. Load advisory data from PostgreSQL: query `travel_advisories` table, build dict of `{country_iso: max_level}` (take max level across sources per country)
     d. Get canonical country list: `from src.seeding.fips import get_sovereign_isos`
     e. For each country in sovereign ISOs:
        - Count GDELT events (source='gdelt') in 90-day window with exponential decay weighting
        - Count ACLED events (source='acled') + sum fatalities (not available in schema -- use event_count * 2 as proxy, or 0)
        - Get advisory level from advisory dict (default 1)
        - Compute average Goldstein score (decay-weighted)
        - Get population via `from src.seeding.population import get_population`
        - Compute baseline risk via `from src.seeding.baseline_risk import compute_baseline_risk`
        - Determine disputed flag: `country_iso in {"XK", "TW", "PS", "EH"}`
     f. Compute heatmap hex bins: `from src.seeding.heatmap_binner import bin_events_to_h3`
     g. Compute bilateral arcs: `from src.seeding.arc_builder import extract_bilateral_arcs`
     h. Load previous baseline risks from PostgreSQL (for risk delta computation)
     i. Compute risk deltas: `from src.seeding.risk_delta import compute_risk_deltas`
     j. Write ALL results to PostgreSQL in a single async session:
        - DELETE all rows from baseline_country_risk, then INSERT new rows (full replace)
        - DELETE all rows from heatmap_hexbins, then INSERT new rows
        - DELETE all rows from country_arcs, then INSERT new rows
        - DELETE all rows from risk_deltas, then INSERT new rows
        - All within one session.commit()
     k. Return counts: `{"countries": N, "hexbins": N, "arcs": N, "deltas": N}`
   - NO large static dicts. All country iteration via pycountry + get_sovereign_isos().
